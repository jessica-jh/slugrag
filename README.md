# SlugRAG: Evaluation Code and Results

This repository contains the code and artifacts used to **evaluate SlugRAG**, our **multi-turn Retrieval-Augmented Generation (RAG)** system, **after the system was built**.

Specifically, it includes:
1) utilities to **validate JSONL files** used in our evaluation pipeline,
2) scripts to **compute retrieval metrics** (e.g., nDCG and Recall) from SlugRAG retrieval outputs, and
3) optional scripts to run the **LLM-based judging components** used in our evaluation of generated responses.

> Note: This repository focuses on the **evaluation pipeline and the evaluation outputs/results**, rather than re-building SlugRAG itself.
